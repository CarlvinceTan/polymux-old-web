# DESIGN.md — web

Living reference for UI/UX decisions and visual conventions. Follow these rules when building or modifying components.

## Icon Buttons

- No grey (or any colour) background highlight on hover or active/pressed states.
- The only visual feedback is the icon itself becoming darker.
- This applies to all standalone icon buttons (e.g. toolbar actions, inline controls). Toggle buttons with an explicit selected state may use a subtle indicator, but still no background fill on hover/press.
