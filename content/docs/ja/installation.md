# インストール

Polymuxはブラウザだけで完結して利用できます。インストールは不要です。ネイティブアプリやブラウザ拡張機能を使うと、ホスト型のブラウザではなくローカルブラウザを動かすなど、追加の機能が利用できます。

## ウェブアプリ

[polymux.com](https://polymux.com)のウェブアプリは常に最新バージョンです。最新のChromium、Firefox、Safariのいずれでも動作します。インストールするものはなく、サインインすればワークスペース、セッション、ワークフローがすぐに利用できます。

## ブラウザ拡張機能

拡張機能を使うと、Polymuxセッションがサーバーのホスト型Chromiumではなく、**ローカルブラウザ** のタブを操作できるようになります。次のような場合に便利です。

- 既存のログインCookieを使うサイトを利用したい場合。
- ホスト型ブラウザから到達できないプライベートネットワークにアクセスしたい場合。
- 特定のブラウザプロファイル、拡張機能リスト、デバイスフィンガープリントを使いたい場合。

インストールするには、[Install page](/install-apps)を開いてブラウザを選択してください。拡張機能がペアリングを要求したら、任意のタブでPolymuxにサインインしてください。ペアリングは自動的に行われ、ポップアップに _Connected_ と表示されます。

拡張機能は接続中もまったく受動的です。Polymuxセッションが`?mode=extension`になっているときだけ動作します。ポップアップからいつでも取り消せます。

## デスクトップアプリ

macOS、Windows、Linux用のネイティブアプリは、ブラウザタブなしでPolymuxの全機能を提供します。現在プライベートベータ中です。プラットフォーム向けのビルドが利用可能になった際に通知を受け取るには、[Install page](/install-apps)で登録してください。

デスクトップアプリは必須ではありません。このドキュメントのすべての機能はウェブアプリで動作します。

## モバイルアプリ

iOSおよびAndroidアプリはロードマップに含まれています。現時点では、ウェブアプリがレスポンシブでモバイルブラウザでも動作しますが、ライブビューポートはデスクトップで最も快適にストリーミングされます。

## システム要件

| 表面 | 要件 |
| --- | --- |
| ウェブアプリ | 過去24か月以内にリリースされた任意のブラウザ |
| 拡張機能 | Chrome、Edge、Brave、またはChromium 119+ |
| デスクトップ | macOS 13+、Windows 10+、または過去3年以内のLinuxディストロ |
| ネットワーク | ポート 443 / 8080 でのWebRTCおよびWebSocketのエグレス |

ネットワークがWebRTCをブロックしている場合、ライブビューポートは数秒後により低速なポーリングストリームにフォールバックします。それ以外の機能は引き続き動作します。
