# 简介

Polymux 是一个用于构建、运行和分享 AI 智能体的平台,这些智能体可以代表你在网络上完成工作。智能体会驱动真实的浏览器会话、相互交接任务、将密钥存放在保险库中、并可回放其运行过程——所有这一切都在你的团队所控制的工作空间内完成。

本文档分为两部分。**用户指南**会引导你安装 Polymux、完成初始配置并在日常工作中使用本产品。**开发者指南**会讲解如何构建并发布打包的工作流和连接器,使其他人能够一键安装使用。

如果你以前没有使用过 Polymux,请从 [快速入门](/documentation/quickstart) 开始。如果你来此查阅 API 参考,可以直接跳转到 [API 概览](/documentation/api-overview)。

## Polymux 为你提供什么

Polymux 围绕三种基本概念构建:

- **会话(Sessions)** —— 与一个智能体之间的一次对话,背后由一个真实的浏览器支撑。会话会将每一次点击、滚动和表单填写实时推送到你的仪表板。
- **工作流(Workflows)** —— 由智能体、视口和工具组成的有向图。工作流支持版本管理、可被调度,也可以打包发布到市场。
- **工作空间(Workspaces)** —— 你的团队共享的容器,容纳工作流、保险库条目、文件和账单。成员需通过邀请加入并被分配角色。

这三个概念会在后续文档中反复出现。如果某个章节直接使用却未加解释,请回到本页查阅。
