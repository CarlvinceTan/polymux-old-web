# 常见问题

按主题分组的常见问题。如果你的问题不在此处,[论坛](/forum) 是最快的提问之处。

## 账户和账单

### 如何更改我的电子邮件地址?

从仪表板打开 **Settings → Account → Email**。在切换生效之前,你需要从旧地址和新地址同时确认更改。

### 我可以在工作空间之间转移工作流吗?

不能直接转移。推荐的路径是将工作流发布为 [打包工作流](/documentation/plugin-overview),并将其安装到目标工作空间。这会迁移提示词、工具和保险库引用,但会重置运行历史。

### 如果我取消订阅,我的数据会怎样?

免费层级的数据将无限期保留。付费方案在取消时会降级为免费层级,这可能会让用量超出免费配额。Polymux 在删除任何超额数据以将你恢复到配额内之前,会在应用内通知你 30 天。

## 会话和工作流

### 为什么我的会话断开了?

会话保留在服务器上保持活跃。如果你的本地浏览器标签页关闭或失去网络连接,会话会继续运行,你可以从侧边栏重新接入。如果会话本身结束,你会在聊天框顶部看到状态码——`idle_timeout`、`budget_exceeded` 或 `error` 是常见的几种。

### 一个会话可以运行多久?

并没有硬性的实际时钟限制,但每个会话都有其工作流上设定的令牌预算。当预算耗尽时,会话会暂停并请求批准后才继续。默认预算足以应对大多数多小时的浏览器任务。

### 为什么实时视口是黑色的?

通常有三个原因:

- **WebRTC 在你的网络上被阻止** ——视口会在几秒后回退到较慢的轮询流。检查视口角上的连接图标;如果它显示黄点,你就处于回退模式。
- **智能体尚未进行导航。** 在浏览器加载第一个页面之前,视口会保持空白。
- **你正在以 `?mode=extension` 运行**,但扩展未配对。请打开扩展弹窗并检查状态徽章。

### 权限错误

如果某个工作流因权限错误而拒绝运行,最常见的原因是你的工作空间角色不允许运行工作流——viewer 无法启动运行。请让管理员将你升级为 **Member**。

## 保险库与密钥

### 模型能看到我的密码吗?

不能。保险库的值会直接被注入页面操作或工具调用中。模型只会被告知使用了某个密钥,而绝不会看到实际的字符。完整说明请参见 [保险库基础](/documentation/vault#how-agents-access-the-vault)。

### 你们使用何种加密?

保险库条目使用 AES-256-GCM 通过每个工作空间各自的密钥进行静态加密。工作空间密钥本身又会被托管 KMS 中保存的根密钥包装。我们绝不会记录或存储解密后的值。

### 我可以导出保险库条目吗?

目前没有批量导出功能。每个条目都可以在保险库页面单独显示和复制。导出工具在路线图中;跟踪问题在 [论坛](/forum)。

## 插件与开发者相关问题

### 什么是插件?

_插件_ 是一个打包的工作流以及它的连接器——其他人将它一键安装到自己工作空间所需要的一切。参见 [插件概览](/documentation/plugin-overview)。

### 如何发布插件?

打开你想要发布的工作流,前往 **Publish** 选项卡,填写列表字段,然后提交审核。审核通常在两个工作日内完成。完整流程参见 [发布插件](/documentation/publishing)。

### 插件之间可以读取彼此的数据吗?

不能。每个插件都在自己的会话中运行,具有自己范围限定的保险库访问权限。安装在同一工作空间内的两个插件,无法看到彼此的保险库读取、文件或会话历史。

## 校验下载

桌面安装程序和扩展 `.zip` 归档文件已签名。每个版本的 SHA-256 会在 [安装页面](/install-apps) 上紧邻下载链接处列出。例如,要校验一个 macOS `.dmg`:

```sh
shasum -a 256 Polymux-1.0.0-universal.dmg
```

将输出与安装页面上的值对照。如果不匹配,请不要运行该安装程序——通过 [联系页面](/contact) 与我们联系。

## 仍然无法解决?

如果上述内容都没有回答你的问题:

1. 搜索 [论坛](/forum) ——常见问题通常在那里有讨论。
2. 如果你正在以编程方式集成,可浏览 [API 概览](/documentation/api-overview)。
3. 从 [联系页面](/contact) 发邮件联系支持。请附上你的 workspace ID 和 session ID(可在 URL 中看到),以便我们在日志中找到你的运行。
