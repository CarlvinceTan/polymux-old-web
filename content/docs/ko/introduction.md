# 소개

Polymux는 여러분을 대신해 웹을 다루는 AI 에이전트를 구축하고, 실행하고, 공유할 수 있는 플랫폼입니다. 에이전트는 실제 브라우저 세션을 직접 조작하고, 서로에게 작업을 인계하며, 비밀 정보를 vault에 저장하고, 실행 과정을 재생합니다. 이 모든 것이 여러분의 팀이 통제하는 워크스페이스 안에서 이루어집니다.

이 문서는 두 부분으로 나뉘어 있습니다. **사용자 가이드**는 Polymux 설치, 초기 설정, 그리고 일상적인 사용 방법을 안내합니다. **개발자 가이드**는 패키지화된 워크플로와 연결을 구축하고 게시하여 다른 사람들이 한 번의 클릭으로 설치할 수 있도록 만드는 방법을 설명합니다.

Polymux를 처음 사용하신다면 [Quickstart](/documentation/quickstart)부터 시작하세요. API 레퍼런스를 찾고 계신다면 바로 [API overview](/documentation/api-overview)로 이동하시면 됩니다.

## Polymux가 제공하는 것

Polymux는 세 가지 기본 요소를 중심으로 구축되었습니다.

- **세션** — 라이브 브라우저로 뒷받침되는 단일 에이전트와의 대화. 세션은 모든 클릭, 스크롤, 폼 입력을 여러분의 대시보드로 스트리밍합니다.
- **워크플로** — 에이전트, 뷰포트, 도구로 구성된 방향성 그래프. 워크플로는 버전이 관리되며, 스케줄링하거나 마켓플레이스용으로 패키지화할 수 있습니다.
- **워크스페이스** — 여러분의 팀이 사용하는 워크플로, vault 항목, 파일, 결제를 담는 공유 컨테이너. 멤버는 초대로 추가되고 역할이 부여됩니다.

이 세 가지 개념은 나머지 문서 곳곳에서 등장합니다. 어떤 섹션이 이 개념들을 별도의 설명 없이 사용한다면, 다시 돌아와 참고할 페이지가 바로 여기입니다.

## 새로운 소식

이 문서 사이트 자체도 진행 중인 작업입니다. 사이드바에서 _Draft_로 표시된 페이지는 아직 미완성 상태입니다. 어떤 주제가 완전히 누락되어 있다면 [forum](/forum)이 가장 빠른 문의 경로입니다.

## 이 문서를 읽는 방법

좌측 사이드바는 페이지를 섹션별로 그룹화합니다. 우측 레일은 현재 페이지에서 여러분의 위치를 추적하며 헤딩 간 이동을 가능하게 합니다. 모든 페이지는 읽기 순서상 이전 페이지와 다음 페이지로 연결되는 링크로 끝나므로, 사이드바로 돌아가지 않고도 가이드 전체를 처음부터 끝까지 차례로 살펴볼 수 있습니다.

Polymux는 활발히 개발되고 있습니다. 제품이 변경될 때마다 이 페이지들을 갱신하므로, 오래된 가이드를 보고 계신다면 다시 확인하실 가치가 있습니다.
